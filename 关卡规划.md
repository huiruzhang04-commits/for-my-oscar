## 🌿 世界1：草地王国（L1-L5）

**句型**：It is a/an X. / Is it a X? / It is not a X.
**新机制**：跳跃、顶砖块、蘑菇变大、踩敌人、星星无敌、管道

### L1 · Aa（4词）· 无Boss

apple 🍎 / cat 🐱 / bat 🦇 / bag 🎒

**流程**（~70秒）：
1. 开场学操作（0-10秒）：顶砖块→蘑菇→管道
2. 🍎 apple（10秒）：I want to eat an ___.
3. 游戏段A：学踩敌人（10-30秒）
4. 🐱 cat（30秒）：The ___ is sleeping on the sofa.
5. 游戏段B：连续跳跃+星星（30-50秒）
6. 🦇 bat（50秒）：A ___ can fly at night.
7. 游戏段C：综合练习（50-70秒）
8. 🎒 bag（70秒）：I put my book in the ___.
9. 通关画面（无Boss，直接过关）

### L2 · Bb（4词）+ Boss战

baby 👶 / banana 🍌 / ball ⚽ / boy 👦

**流程**（~65秒+Boss）：
1. 快速回顾（0-8秒）：砖块+踩敌人
2. 👶 baby：It is a ___.
3. 🍌 banana：It is a ___.
4. ⚽ ball：It is a ___.
5. 👦 boy：It is a ___.
6. **库巴Boss战**（3题）：
   - 题目用L2的例句，选项从L1+L2的8个词中选
7. 通关画面

### L3 · Cc（5词）+ Boss战

cow 🐄 / car 🚗 / cake 🎂 / clock 🕐 / candy 🍬

**Boss战**（3题）：从L3例句中出题，选项混入L1~L3词汇

### L4 · Dd（5词）+ Boss战

dad 👨 / duck 🦆 / desk 🪑 / door 🚪 / dog 🐕

**Boss战**（3题）：从L4例句中出题，选项混入L1~L4词汇

### L5 · Ee（3词）+ Boss战

egg 🥚 / elephant 🐘 / bed 🛏️

**世界1终关Boss**（3题）：综合L1~L5例句

## 每关通用结构

```
[开场回顾] → [词1] → [跑酷段] → [词2] → [跑酷段] → [词3] → [跑酷段] → [词4] → [Boss战] → [通关]
```

> L1例外：无Boss，[词4]后直接[通关]

### Boss战规则
- **题目数**：3题（L26为5题）
- **题目来源**：当前关卡的例句
- **选项来源**：当前关卡单词 + 前几关已学单词混选
- **答题方式**：英文句子填空，4个英文单词选项，附中文释义
- **答对**：库巴后退+金币雨
- **答错**：库巴前进（不扣命，重选）
- **全对**：库巴飞走+通关

### 单词卡结构

**普通题**（每关4-5题）：
- 图标 + 英文单词
- 例句填空（每词专属例句）
- 4个中文选项（1正确+3同关卡错误）

**Boss战**（每关末尾）：
- 英文句子填空
- 中文释义
- 4个英文单词选项
