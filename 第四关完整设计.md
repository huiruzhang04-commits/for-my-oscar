# 马里奥英语游戏 · 第四关完整设计（L4）

## 世界：🌿 草地王国
## 字母：Dd
## 单词：dad, duck, desk, door, dog
## 句型：It is a/an X. / Is it a X? / It is not a X.

## 与前三关的差异
- 新机制：**管道探索**（进入管道可以发现隐藏区域，有额外金币和单词卡）
- 单词数5个，与L3相同
- Boss选项范围：L1~L4全部词汇（4+4+5+5=18词）
- 总时长：约80秒 + Boss战3题

### 库巴Boss战（75秒）· 3题连续

库巴从屏幕右边跳出来，挡住去路。弹出选词填空卡片，需要连续答对3题才能通关。

**题目从L1~L4的例句中抽取，选项从四个关卡的18个词中混选。**

**第1题（简单）**
- 句子：It is a ___.
- 选项：dad / duck / apple / cat
- 正确：dad
- 来源：L4例句

**第2题（中等）**
- 句子：The ___ is sleeping on the sofa.
- 选项：dog / desk / door / duck
- 正确：cat
- 来源：L1例句

**第3题（稍难）**
- 句子：It is a ___.
- 选项：door / cow / car / clock
- 正确：door
- 来源：L4例句

**答对1题：** 库巴往后退一步+金币雨
**答对2题：** 库巴再退一步+更大金币雨
**答对3题：** 库巴被打败飞走+撒花+通关！
**答错：** 库巴往前走一步（不惩罚，重选）
