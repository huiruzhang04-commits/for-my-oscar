# 马里奥英语游戏 · 世界4完整设计（L17-L21）· 天空之城

## ☁️ 世界4：天空之城
## 字母：Qq~Uu
## 句型：Can X do Y? / This is a X. / She is a X. / Do you have…?

## 天空专属元素
- ☁️ 消失云平台：踩上去3秒后碎裂，需要快速跳走
- 💨 风力：特定路段有风，把马里奥往左/右推
- 🐦 飞鸟：替代小鱼，水平飞行，可踩
- ☁️ 云怪：云形状敌人，上下浮动，可踩
- ⭐ 浮岛：替代砖块，顶出金币/蘑菇/星星

## 世界4 Boss战机制
骷髅管道进入 → 库巴在云上高台 → 跳到正确云石台 → 云砖块砸库巴 → 金币雨
答错惩罚：飞鸟/云怪攻击
