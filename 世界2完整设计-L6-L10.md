# 马里奥英语游戏 · 世界2完整设计（L6-L10）· 洞穴王国

## 🕳️ 世界2：洞穴王国
## 字母：Ff~Jj
## 句型：This is a X and that is a Y. / These are… / Those are…

## 洞穴专属元素
- 🦇 蝙蝠：空中浮动敌人，不能踩，只能躲
- 🕷️ 蜘蛛：天花板垂下来的敌人，占高处，跳高会碰
- 🔥 火把：标记安全区，旁边有金币
- 🪨 石笋：地面障碍，不能踩，要跳过（龟壳可以撞碎）
- 🌑 暗区：部分路段只有火把光，远处看不清

## 世界2 Boss战机制（每关统一）

**① 管道入口（3秒）**
- 关卡最后一段出现骷髅标记管道
- 跳进去 → 黑屏过渡 → 进入Boss洞穴

**② Boss洞穴场景**
- 比普通洞穴更暗，只有火把光
- 库巴站在中央高台上
- 天花板有三个"？"砖块，每个砖块对应一道题

**③ 答题打库巴（核心）**
- 库巴说话出题（语音+文字）
- 三个选项分布在地面上的三个石台上
- 马里奥跳到正确石台 → 砖块掉下来砸库巴
- 答错 → 库巴大笑，掉一只蝙蝠过来攻击（扣一格血）
- 三题全对 → 库巴被砸三次，掉下高台

**④ 金币雨（4秒）**
- 库巴掉下后变成一堆金币散落
- 马里奥跑过去收集，金币自动飞过来（磁吸效果）
- 收集完 → 通关画面
