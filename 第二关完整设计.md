# 马里奥英语游戏 · 第二关完整设计（L2）

## 世界：🌿 草地王国
## 字母：Bb
## 单词：baby, banana, ball, boy
## 句型：It is a/an X. / Is it a X? / It is not a X.

## 与第一关的差异
- 新增机制：**移动平台**（会左右移动的砖块，需要判断跳的时机）
- 单词卡句子升级：从纯填空变成 **It is a/an X.** 句型
- 敌人节奏稍快
- 总时长：约75秒

---

## 完整流程

### 开场（0-8秒）· 回顾操作

| 时间 | 元素 | 说明 |
|------|------|------|
| 0-3秒 | 空旷地面，自动往前走 | 热身，进入状态 |
| 3-5秒 | 一个"？"砖块，顶出金币 | 回忆操作：砖块有奖励 |
| 5-8秒 | 一个板栗怪走过来，踩它 | 回忆操作：跳起来踩敌人 |

> 比第一关开场短2秒，因为孩子已经会操作了，不需要教蘑菇变大（第一关教过了）

### 库巴Boss战（65秒）· 3题连续

库巴从屏幕右边跳出来，挡住去路。弹出选词填空卡片，需要连续答对3题才能通关。

**题目从第一关和第二关的例句中抽取，选项从两个关卡的8个词中混选。**

**第1题（简单）**
- 句子：I want to eat an ___.
- 选项：apple / baby / ball / bag
- 正确：apple
- 来源：第一关例句

**第2题（中等）**
- 句子：It is a ___.
- 选项：banana / bat / cat / boy
- 正确：banana
- 来源：第二关例句

**第3题（稍难）**
- 句子：The ___ is sleeping on the sofa.
- 选项：boy / bag / cat / baby
- 正确：cat
- 来源：第一关例句

**答对1题：** 库巴往后退一步+金币雨
**答对2题：** 库巴再退一步+更大金币雨
**答对3题：** 库巴被打败飞走+撒花+通关！
**答错：** 库巴往前走一步（不惩罚，重选）
